Some users don't want a background graphic. It is possible to draw an own background. For instance with a gradient paint.
In the middle there may be a light spot which should be a little bit bigger than the level graphics.
Example how it may look like (without the level graphics):


For testing you can go to: File MainBoardDisplay.java, method: newGraphicSizeCalculation.
